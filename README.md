### VisualStudio/C#

* <b>meth:</b>
Cria um novo método sem parâmetros.
```
public void MyMethod()
{

}
```

* <b>meth(1-4):</b>
Cria um novo método, onde o número de 1 a 4 representa a quantidade de parâmetros especificada. Ex: meth1 cria um método com 1 parâmetro, e meth4 (máximo) cria um método com 4 parâmetros.
```
public void MyMethod(string param1, string param2, string param3)
{

}

```
