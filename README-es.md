A partir de ahora voy a comenzar a compartir también algunos code snippets que yo uso para facilitar mi trabajo.

### VisualStudio/C#

* <b>meth:</b>
Crea un nuevo método sin parámetros.
```
public void MyMethod()
{

}
```

* <b>meth(1-4):</b>
Crea un nuevo método, donde el número de 1 a 4 representa la cantidad de parámetros especificada. Ej: meth1 crea un método con 1 parámetro, y meth4 (máximo) crea un método con 4 parámetros.
```
public void MyMethod(string param1, string param2, string param3)
{

}

```
